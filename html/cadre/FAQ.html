Foire aux questions

Comment utiliser Intérimédical ?

1. Que faire lorsque je suis perdu ?

Veuillez consulter cette FAQ attentivement avant de contacter le support d'intérimédical.

2. Comment la plateforme fonctionne ?

En tant que cadre, le contexte de la recherche d'intérimaire est déjà pré-défini.
Cliquez sur la section demande d'intérimaire pour solliciter une intérimaire.

3. Quelles sont les étapes pour recruter un intérimaire ?

Les voici :

  1. Demander un intérimaire via l'interface et renseigner les informations ;
  2. Une notification vous préviendra à chaque intérimaire postulant ;
  3. Valider l'intérimaire s'il correspond à vos besoins ;
  4. Une fois la mission terminé, il vous faudra valider leur présence et attribuer une note

4. Comment se passe la facturation ?

La société Intérimédical va s'occuper de la facturation.

5. Todo

Longtemps, je me suis couché de bonne heure.
Parfois, à peine ma bougie éteinte, mes yeux se fermaient si vite que je n’avais pas le temps de me dire : « Je m’endors.».

Que faire en cas de problème de connexion ?

1. Le message "mot de passe incorrect" s'affiche.

Vérifiez votre mot de passe collé sur un post-it sur l'écran.

2. Une erreur 404 apparait.

Vérifiez votre connexion internet, si cela ne résoud pas le problème. Contactez votre administrateur système.

3. Le message "Votre compte a expiré" apparait.

Il est possible que votre compte n'ait pas été renouvelé. Veuillez contacter le responsable pour résoudre le problème.

4. Rien ne s'affiche.

Dans ce cas, vous pouvez rafraichir la page en appuyant sur la touche 5. Ou ouvrir votre fenêtre.

5. Un pokémon sauvage apparait.

Tenez, prenez ces cachets.
